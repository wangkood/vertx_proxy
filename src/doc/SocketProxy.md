## socket代理协议


### 握手协调阶段
```bash
> 客户端向服务器发送请求
+----+---------+--------+
|VER |NMETHODS |METHODS |
+----+---------+--------+
|1   |1        |1 to 225|
+----+---------+--------+
> 服务器响应
+----+-------+
|VER |METHOD |
+----+-------+
|1   |1      |
+----+-------+
```

### 客户端发送指令
```bash
> 客户端向服务器发送请求
+----+----+----+------+---------+---------+
|VER |CMD |RSV |ATYPE |DST.ADDR |DST.PORT |
+----+----+----+------+---------+---------+
|1   |1   |1   |1     |var      |2        |
+----+----+----+------+---------+---------+
> 服务器响应
+----+----+----+-----+---------+---------+
|VER |REP |RSV |ATYP |BND.ADDR |BND.PORT |
+----+----+----+-----+---------+---------+
|1   |1   |1   |1    |var      |2        |
+----+----+----+-----+---------+---------+
```
